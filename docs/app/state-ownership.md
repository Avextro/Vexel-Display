# State Ownership

## Global State

Owned by app/state

Contains:

- User session - perssistant
- Theme preferences - perssistant
- Application Settings - perssistant
- Active LED panel

## Editor State

Owned by features/editor

Contains:

- Current animation/image
- Current tool

## Component State

Owned locally

Contains:

- Inputs
- Animations
- Visibility
- Interaction
